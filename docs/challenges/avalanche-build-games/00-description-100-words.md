Stabletown CityChain Kit is a repeatable Avalanche package for civic quest economies. In strict mode it deploys CityChain contracts to Fuji, runs three quest loops (`onboarding`, `merchant_spend`, `community_event`), executes merchant redemption settlement, and emits deploy/flow receipts with explorer links. Judge Mode (`/citychain/judge`) unifies the live demo: preflight, loops, settlement, tx-intent finality polling, and required proof-slot attachment. It also captures validator/governance state and embedded-wallet session evidence through first-party APIs. Optional local L1 bootstrap signal is exported as an artifact. Non-strict mode stays deterministic for CI. Every write remains tenant-scoped, idempotent, audited, and outboxed while canonical ledger behavior remains intact.
